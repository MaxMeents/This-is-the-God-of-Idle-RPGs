The Magic Behind the Scenes

Ever wonder how these pictures move? It's just simple code! Here is exactly how we did it:

1 First, make a "Window"

We use a <div> tag. It's like a square hole in a wall where we see the art.

HTML
<div id="my-window"></div>
2 Frame the Window with CSS

We tell the window exactly how big it should be and put our long strip of art behind it.

CSS
#my-window {
  width: 64px;
  height: 64px;
  background: url('Blue White Butterfly Flying Forward_64x64_sheet.png');
}
3 The "Magic Timer"

This code slides the strip of art behind the window very fast. It feels like it's moving!

JAVASCRIPT
// 1. How many columns are on our sheet?
const cols = 6;
// 2. Keep track of which frame to show
let frame = 0;
// 3. Change frame every 14 milliseconds
setInterval(() => {
  const x = (frame % cols) * 64;
  const y = Math.floor(frame / cols) * 64;
  document.getElementById('my-window').style.backgroundPosition = `-${x}px -${y}px`;
  frame = (frame + 1) % 32;
}, 14);
